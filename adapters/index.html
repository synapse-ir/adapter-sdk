    Contribute an adapter¶

    1. Install: pip install synapse-adapter-sdk
    2. Write your adapter following the first adapter guide
    3. Validate: synapse-validate --adapter your_module.YourAdapter --all-fixtures
    4. Open a PR to github.com/synapse-ir/adapters

    See BOUNTIES.md for models where adapters are wanted.
